# Bookmarks.cards

Bookmark.cards is a bookmarket collection tool in its simpliest form.

## BrandsCSS

```bash
npm i --save-dev colorthief
npm i --save-dev jimp
npm i --save-dev get-image-colors
npm run generateBrandsCSS
```

## Test

```bash
cd /your/project/path
npm install cypress --save-dev
npm init
npm run cy:open
npm run cy:test
```
